# PyQt5 Practice

## Set up
1. Create a new enviroment
```
conda create -n qt_env python=3.8
```
2. Activate the new enviroment
```
activate qt_env
```
3. Install PyQt5
```
pip install pyqt5
pip install pyqt5-tools
```
4. Check
```
import PyQt5
```
